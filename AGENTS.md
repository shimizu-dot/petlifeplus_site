# AGENTS.md

## Global Rules (Must Follow)
- 不要な確認質問をしない。指示がある作業はそのまま実行する。
- 作業対象外のセクション・ファイルを触らない。
- 指摘されたレイアウト崩れは、対象要素だけを最小差分で修正する。
- 編集後は必ず `git diff` で意図しない変更がないか確認する。

## HTML/SVG Editing Rules
- 「実装準拠ポイント:」は文書として記載し、画像外に置く。
- 「実装準拠ポイント:」の直後で改行する（`実装準拠ポイント:<br>...`）。
- 注釈の位置指定がある場合は厳守する（例: 画像のすぐ下、画像内など）。
- 線と文字が重ならないように座標を調整する。
- フォントサイズ変更は指示がない限り他要素へ波及させない。

## PetEntity Section Specific
- 作業中は `design/class-diagram.html` の PetEntity ブロックのみ編集する。
- User.java ブロックには一切追記しない。

## Post-Edit Checklist
1. `実装準拠ポイント:` の直後で改行している。
2. 説明文と罫線が重なっていない。
3. 余白が過剰になっていない。
4. User.java 側へ混入していない。
