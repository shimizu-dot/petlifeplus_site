Broadcast の仕組み
  1. ユーザーが LINE で Bot を友達追加する
  2. 管理者がこのページからメッセージを送信する
  3. 友達追加しているユーザー全員に自動で届く（ユーザーIDの個別登録は不要）
💡 Broadcast は LINE 公式アカウント（Messaging API チャンネル）の全フォロワーが対象です。 フォロワー数は LINE Official Account Manager で確認できます。
