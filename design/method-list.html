HTTP メソッドオーバーライド 必須
Thymeleaf フォームで PATCH / DELETE を擬似的に送信する仕組み
項目内容
フィルターSpring の HiddenHttpMethodFilter（SecurityConfig で有効化）
使用方法Thymeleaf フォームに <input type="hidden" name="_method" value="PATCH"> を埋め込む
対象メソッドPATCH（update）、DELETE（delete）を使用する全コントローラー
理由HTML フォームは GET/POST しか送信できないため、Spring の HiddenHttpMethodFilter でサーバー側で変換する

デフォルト page1（1 以下は 1 に補正）
デフォルト size20（1〜100 の範囲に補正）
offset 計算offset = (page - 1) * size
